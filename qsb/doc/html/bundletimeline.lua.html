Module bundletimeline

Dieses Bundle ermöglicht es Aktionen in einer zeitlichen Abfolge zu gliedern.

Die Bezeichnung "Time Line" (Zeitstrahl) wird z.B. in der Filmindustrie verwendet und beschreibt einen Ablauf von aufeinander folgenden Ereignissen. Diese Ereignisse sind zeitlich gegliedert.

Du kannst mit der Implementation eines solchen Zeitstrahls Lua-Funktionen im Abstand einer oder mehrerer Sekunden ausführen lassen. Ein solcher Zeitstrahl verfügt über eine ID, mit der er angesprochen werden kann. So kann er abgebrochen, pausiert, fortgesetzt und neu gestartet werden.

Zeitstrahl erstellen
Eine Abfolge von Aktionen definieren, die zu gewissen Zeitpunkten ausgelöst werden.

Anwenderfunktionen

QSB.TimeLine:IsRunning (_ID) Prüft, ob der Zeitstrahl noch nicht durchgelaufen ist.
QSB.TimeLine:Restart (_ID) Startet einen Zeitstrahl erneut.
QSB.TimeLine:Resume (_ID) Stößt einen angehaltenen Zeitstrahl wieder an.
QSB.TimeLine:Start (_Description) Startet einen Zeitstrahl.
QSB.TimeLine:Yield (_ID) Hält einen Zeitstrahl an.


Anwenderfunktionen

QSB.TimeLine:IsRunning (_ID)
Prüft, ob der Zeitstrahl noch nicht durchgelaufen ist.

Parameters:

  • _ID number ID des Zeitstrahl

Returns:

    boolean Zeitstrahl ist aktiv

Usage:

    local IsRunning = QSB.TimeLine:IsRunning(MyTimeLine);
QSB.TimeLine:Restart (_ID)
Startet einen Zeitstrahl erneut. Ist der Zeitstrahl noch nicht beendet, beginnt er dennoch von vorn.

Parameters:

  • _ID number ID des Zeitstrahl

Usage:

    QSB.TimeLine:Restart(MyTimeLine);
QSB.TimeLine:Resume (_ID)
Stößt einen angehaltenen Zeitstrahl wieder an.

Parameters:

  • _ID number ID des Zeitstrahl

Usage:

    QSB.TimeLine:Resume(MyTimeLine);
QSB.TimeLine:Start (_Description)
Startet einen Zeitstrahl. Ein Zeitstrahl hat Stationen, an denen eine Aktion ausgeführt wird. Jede Station muss mindestens eine Sekunde nach der vorherigen liegen.

Jede Aktion eines Zeitstrahls erhält die Table des aktuellen Ereignisses als Argument. So können Parameter an die Funktion übergeben werden.

Parameters:

  • _Description table Beschreibung

Returns:

    number ID des Zeitstrahls

Usage:

    MyTimeLine = QSB.TimeLine:Start {
    {Time = 5, Action = MyFirstAction},
    -- MySecondAction erhält "BOCKWURST" als Parameter
    {Time = 15, Action = MySecondAction, "BOCKWURST"},
    -- Inline-Funktion
    {Time = 30, Action = function() end},
}

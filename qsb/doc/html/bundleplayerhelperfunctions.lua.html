Module bundleplayerhelperfunctions

Mit diesem Bundle kommen einige Funktionalitäten zur Steuerung von spielerbezogenen Features.

Anwenderfunktionen

API.AllowFestival (_PlayerID) Erlaubt ein normales Fest und gibt die Technologie frei.
API.ForbidFestival (_PlayerID) Verbietet ein normales Fest und sperrt die Technologie.
API.GetControllingPlayer () Gibt die ID des kontrollierenden Spielers zurück.
API.SetControllingPlayer (_OldID, _NewID, _NewName, _RetainKnight) Wechselt die Spieler ID des menschlichen Spielers.
API.SetEarningsOfPlayerCity (_PlayerID, _Earnings) Setzt die Einnahmen für alle Stadtgebäude eines Spielers.
API.SetNeedSatisfaction (_Need, _State, _PlayerID) Setzt den Befriedigungsstatus eines Bedürfnisses für alle Gebäude des angegebenen Spielers.
API.StartCityUpgradeFestival (_PlayerID) Startet ein Beförderungsfest für den Spieler.
API.StartNormalFestival (_PlayerID) Startet ein Fest für den Spieler.
API.UndiscoverTerritories (_PlayerID, _TargetPlayerID) Entfernt alle Territorien einer Partei aus der Liste der entdeckten Territorien.
API.UndiscoverTerritory (_PlayerID, _TerritoryID) Entfernt ein Territorium für den angegebenen Spieler aus der Liste der entdeckten Territorien.
API.UnlockTitleForPlayer (_PlayerID, _KnightTitle) Diese Funktion ermöglicht das sichere Entsperren eines gesperrten Titels.


Anwenderfunktionen

API.AllowFestival (_PlayerID)
Erlaubt ein normales Fest und gibt die Technologie frei.

Alias: AllowFestival

Parameters:

  • _PlayerID number Spieler
API.ForbidFestival (_PlayerID)
Verbietet ein normales Fest und sperrt die Technologie.

Alias: ForbidFestival

Parameters:

  • _PlayerID number Spieler
API.GetControllingPlayer ()
Gibt die ID des kontrollierenden Spielers zurück. Der erste als menschlich definierte Spieler wird als kontrollierender Spieler angenommen.

Alias: PlayerGetPlayerID

Returns:

    number PlayerID
API.SetControllingPlayer (_OldID, _NewID, _NewName, _RetainKnight)
Wechselt die Spieler ID des menschlichen Spielers. Die neue ID muss einen Primärritter haben. Diese Funktion kann nicht im Multiplayer Mode verwendet werden.

Alias: PlayerSetPlayerID

Parameters:

  • _OldID number Alte ID des menschlichen Spielers
  • _NewID number Neue ID des menschlichen Spielers
  • _NewName string Name in der Statistik
  • _RetainKnight boolean Ritter mitnehmen
API.SetEarningsOfPlayerCity (_PlayerID, _Earnings)
Setzt die Einnahmen für alle Stadtgebäude eines Spielers. Stadtgebäude können nur Einnahmen zwischen 0 und 100 Talern haben.

Alias: SetPlayerEarnings

Parameters:

  • _PlayerID number Partei oder nil für alle
  • _Earnings number Einnahmen [0 | 100]
API.SetNeedSatisfaction (_Need, _State, _PlayerID)
Setzt den Befriedigungsstatus eines Bedürfnisses für alle Gebäude des angegebenen Spielers. Der Befriedigungsstatus ist eine Zahl zwischen 0.0 und 1.0.

Alias: SetNeedSatisfactionLevel

Parameters:

  • _Need number Bedürfnis
  • _State number Erfüllung des Bedürfnisses
  • _PlayerID number Partei oder -1 für alle
API.StartCityUpgradeFestival (_PlayerID)
Startet ein Beförderungsfest für den Spieler. Ist dieser Typ von Fest für den Spieler verboten, wird er automatisch erlaubt.

Alias: StartCityUpgradeFestival

Parameters:

  • _PlayerID number Spieler
API.StartNormalFestival (_PlayerID)
Startet ein Fest für den Spieler. Ist dieser Typ von Fest für den Spieler verboten, wird er automatisch erlaubt.

Alias: StartNormalFestival

Parameters:

  • _PlayerID number Spieler
API.UndiscoverTerritories (_PlayerID, _TargetPlayerID)
Entfernt alle Territorien einer Partei aus der Liste der entdeckten Territorien. Als Nebeneffekt gild die Partei als unentdeckt.

Alias: UndiscoverTerritories

Parameters:

  • _PlayerID number Spieler-ID
  • _TargetPlayerID number Anderer Spieler
API.UndiscoverTerritory (_PlayerID, _TerritoryID)
Entfernt ein Territorium für den angegebenen Spieler aus der Liste der entdeckten Territorien.

Alias: UndiscoverTerritory

Parameters:

  • _PlayerID number Spieler-ID
  • _TerritoryID number Territorium-ID
API.UnlockTitleForPlayer (_PlayerID, _KnightTitle)
Diese Funktion ermöglicht das sichere Entsperren eines gesperrten Titels.

Alias: UnlockTitleForPlayer

Parameters:

  • _PlayerID number Zielpartei
  • _KnightTitle number Titel zum Entsperren

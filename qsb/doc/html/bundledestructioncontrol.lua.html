Module bundledestructioncontrol

Ermöglicht es einzelne Gebäude, Typen oder Kategorien von Gebäuden vor dem Abriss zu schützen.

Ebenso kann jeglicher Abriss von Gebäuden auf einem Territrium komplett unterbunden werden.

Das wichtigste Auf einen Blick:

  • Den Abriss für bestimmte Entities steuern.
    Entities beschützen, Schutz aufheben

Anwenderfunktionen

API.ProtectCategory (_entry) Fügt eine Kategorie hinzu, die nicht abgerissen werden darf.
API.ProtectEntity (_entry) Fügt ein Entity hinzu, dass nicht abgerissen werden darf.
API.ProtectEntityType (_entry) Fügt einen Entitytyp hinzu, der nicht abgerissen werden darf.
API.ProtectTerritory (_entry) Fügt ein Territory hinzu, auf dem nichts abgerissen werden kann.
API.UnprotectCategory (_entry) Entfernt eine Kategorie, die nicht abgerissen werden darf.
API.UnprotectEntity (_entry) Entfernt ein Entity, dass nicht abgerissen werden darf.
API.UnprotectEntityType (_entry) Entfernt einen Entitytyp, der nicht abgerissen werden darf.
API.UnprotectTerritory (_entry) Entfernt ein Territory, auf dem nichts abgerissen werden kann.

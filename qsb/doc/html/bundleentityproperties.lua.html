Module bundleentityproperties

In diesem Bundle werden grundlegende Funktionen zur Abfrage und Manipulation von Entities bereitgestellt.

API.ChangeEntityHealth (_Entity, _Percentage)
Ändert die relative Gesundheit eines Entity.

Hinweis: Die Gesundheit ist immer ganzzahlig zwischen 0 und 100.

Alias: SetHealth

Parameters:

  • _Entity string Scriptname des Entity
  • _Percentage number Relative Gesundheit des Entity
API.GetEntityHealth (_Entity)

Alias: GetHealth

Parameters:

Returns:

    number Relative Gesundheit des Entity

API.GetEntityScale (_Entity)
Gibt den Größenfaktor des Entity zurück.

Der Faktor gibt an, um wie viel die Größe des Entity verändert wurde, im Vergleich zur normalen Größe. Faktor 1 entspricht der normalen Größe.

Alias: GetScale

Parameters:

  • _Entity string Scriptname des Entity

Returns:

    number Größenfaktor des Entity
API.GetMovementTarget (_Entity)
Gibt die Position zurück, zu der sich ein Entity bewegt. Die Position wird als XY-Table zurückgegeben.

Alias: GetMovingTarget

Parameters:

Returns:

    table Zielposition des Entity
API.HurtEntity (_Entity, _Damage)
Verwundet ein Entity um einen absoluten Wert Gesundheit.

Hinweis: Die Anzahl an Gesundheitspunkten richtet sich nach dem jeweiligen Entity und ist immer ganzzahlig!

Alias: HurtEntityEx

Parameters:

  • _Entity string Scriptname des Entity
  • _Damage number Menge an abgezogener Gesundheit

API.SetBuildingOnFire (_Entity, _FireSize)
Setzt ein Gebäude in Brand. Die Feuerstärke bestimmt, wie häftig es brennt.

Hinweis: Im Allgemeinen reicht eine Feuerstärke zwischen 1 und 50.

Alias: SetOnFire

Parameters:

  • _Entity string Scriptname des Entity
  • _FireSize number Relative Gesundheit des Entity

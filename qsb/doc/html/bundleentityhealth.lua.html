Module bundleentityhealth

Dieses Bundle stellt Funktionen bereit, mit denen die Gesundheit von Entities überwacht oder geändert werden kann.

Das wichtigste Auf einen Blick:

  • Gesundheit ermitteln
  • Gesundheit verändern
  • Gebäude in Brand stecken
  • Trigger anlegen

Anwenderfunktionen

API.GetEntityHealth (_Entity) Gibt die Gesundheit des Entity in prozent zurück.
API.ChangeEntityHealth (_Entity, _Percentage) Ändert die Gesundheit des Entity zu dem angegeben Wert in Prozent.
API.SetBuildingOnFire (_Entity, _Strength) Steckt ein Gebäude in Brand.
API.HurtEntity (_Target, _AmountOfDamage, _Attacker) Verwundet ein Entity oder ein Battalion um den angegebenen Wert.
API.AddOnEntityHurtAction (_Function) Erstellt einen neuen Auslöser für Kampfgeschehen.
API.AddOnEntityDestroyedAction (_Function) Erstellt einen neuen Auslöser für zerstörte Entities.
API.AddOnEntityCreatedAction (_Function) Fügt eine Funktion hinzu, die ausgeführt wird, wenn ein Entity erzeugt wird.

Reprisal

Reprisal_SetHealth (_Entity, _Percentage) Ändert die Gesundheit eines Entity.

Reward

Reward_SetHealth (_Entity, _Percentage) Ändert die Gesundheit eines Entity.

Trigger

Trigger_EntityHealth (_Entity, _Percentage) Die Gesundheit eines Entities muss einen bestimmten Wert erreichen.


Anwenderfunktionen

API.GetEntityHealth (_Entity)
Gibt die Gesundheit des Entity in prozent zurück.

Achtung: Nur Siedler (inklusive Millitär und Helden), Belagerungswaffen, Gebäude und Raubtiere haben Gesundheit.

Alias: GetHealth

Parameters:

  • _Entity Angefragtes Entity (Striptname oder ID)

Returns:

    number Gesundheit in Prozent

Usage:

    local Health = GetHealth("hakim");
API.ChangeEntityHealth (_Entity, _Percentage)
Ändert die Gesundheit des Entity zu dem angegeben Wert in Prozent.

Achtung: Nur Siedler (inklusive Millitär und Helden), Belagerungswaffen, Gebäude und Raubtiere haben Gesundheit.

Alias: SetHealth

Parameters:

  • _Entity Entity dessen Gesundheit geändert wird (Striptname oder ID)
  • _Percentage number Gesundheit in Prozent

Usage:

    SetHealth("hakim", 50);
API.SetBuildingOnFire (_Entity, _Strength)
Steckt ein Gebäude in Brand.

Achtung: Nur Gebäude können in Brand gesteckt werden.

Alias: SetOnFire

Parameters:

  • _Entity Gebäude (Striptname oder ID)
  • _Strength number Stärke des Brandes

Usage:

    SetOnFire("headquarters1", 100);
API.HurtEntity (_Target, _AmountOfDamage, _Attacker)
Verwundet ein Entity oder ein Battalion um den angegebenen Wert. Wird ein Battalion verwundet, werden der Reihe nach alle Soldaten verwundet, bis der gesamte Schaden verrechnet wurde oder alle Sodaten tot sind.

Alias: HurtEntityEx

Parameters:

  • _Target Ziel des Schadens (Striptname oder ID)
  • _AmountOfDamage number Menge an Schaden
  • _Attacker (Optional) Angreifendes Entity (Striptname oder ID)

Usage:

    HurtEntityEx("battalion", 250, "hakim");
API.AddOnEntityHurtAction (_Function)
Erstellt einen neuen Auslöser für Kampfgeschehen.

Jede Funktion wird immer dann ausgeführt, wenn ein Entity durch ein anderes verwundet wird.

Alias: AddHurtAction

Parameters:

  • _Function function Funktion, die ausgeführt wird.

Usage:

    API.AddOnEntityHurtAction(SomeFunctionReference);
API.AddOnEntityDestroyedAction (_Function)
Erstellt einen neuen Auslöser für zerstörte Entities.

Jede Funktion wird immer dann ausgeführt, wenn ein Entity durch ein anderes Entity oder durch das Skript zerstört wird.

Alias: AddKilledAction

Parameters:

  • _Function function Funktion, die ausgeführt wird.

Usage:

    API.AddOnEntityCreatedAction(SomeFunctionReference);
API.AddOnEntityCreatedAction (_Function)
Fügt eine Funktion hinzu, die ausgeführt wird, wenn ein Entity erzeugt wird.

Alias: AddSpawnedAction

Parameters:

  • _Function function Funktion, die ausgeführt wird.

Usage:

    API.AddOnEntityCreatedAction(SomeFunctionReference);

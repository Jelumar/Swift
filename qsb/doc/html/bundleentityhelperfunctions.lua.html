Module bundleentityhelperfunctions

Stellt Hilfsfunktionen bereit um die Eigenschaften von Entities zu ermitteln oder zu verändern.

Das wichtigste Auf einen Blick:

  • Eigenschaften von Entities
  • Entities ermitteln

Anwenderfunktionen

API.EntityGetName (_entity) Gibt den Skriptnamen des Entity zurück.
API.EntityGetOrientation (_entity) Gibt die Orientierung des Entity zurück.
API.EntitySetName (_entity, _name) Setzt den Skriptnamen des Entity.
API.EntitySetOrientation (_entity, _ori) Setzt die Orientierung des Entity.
API.GetEntitiesByPrefix (_Prefix) Gibt alle Entities zurück, deren Name mit dem Prefix beginnt.
API.GetEntitiesNearby (_eID, _entities) Ermittelt aus einer liste von Entity-IDs das Entity, dass dem Basis-Entity am nächsten ist.
API.GetEntitiesOfCategoriesInTerritories (_player, _category, _territory) Sucht auf den angegebenen Territorium nach Entities mit bestimmten Kategorien.
API.GetKnightsNearby (_eID, _playerID) Ermittelt den Helden eines Spielers, ders dem Basis-Entity am nächsten ist.
API.GetRelativePosition (_target, _distance, _angle, _buildingRealPos) Errechnet eine Position relativ im angegebenen Winkel und Position zur Basisposition.

User Spase

API.SetResourceAmount (_Entity, _StartAmount, _RefillAmount) Setzt die Menge an Rohstoffen und die durchschnittliche Auffüllmenge in einer Mine.

API.GetEntitiesByPrefix (_Prefix)
Gibt alle Entities zurück, deren Name mit dem Prefix beginnt.

Alias: GetEntitiesNamedWith

Parameters:

  • _Prefix string Präfix des Skriptnamen

Returns:

    table Liste mit Entities

Usage:

    -- Alle Entities mit "entranceCave" -> entranceCave1, entranceCave2, ...
local Result = API.GetEntitiesByPrefix("entranceCave");
API.GetEntitiesNearby (_eID, _entities)
Ermittelt aus einer liste von Entity-IDs das Entity, dass dem Basis-Entity am nächsten ist.

Alias: GetClosestEntity

Parameters:

  • _eID number Basis-Entity
  • _entities table Liste von Entities

Returns:

    number Nächstes Entity

Usage:

    local EntityList = API.GetEntitiesOfCategoriesInTerritories({1, 2, 3}, EntityCategories.Hero, {5, 12, 23, 24});
local Knight = API.GetEntitiesNearby(GetID("IO1"), EntityList);
API.GetEntitiesOfCategoriesInTerritories (_player, _category, _territory)
Sucht auf den angegebenen Territorium nach Entities mit bestimmten Kategorien. Dabei kann für eine Partei oder für mehrere Parteien gesucht werden.

Alias: GetEntitiesOfCategoriesInTerritories

Alias: EntitiesInCategories

Parameters:

  • _player PlayerID [0-8] oder Table mit PlayerIDs (Einzelne Spielernummer oder Table)
  • _category Kategorien oder Table mit Kategorien (Einzelne Kategorie oder Table)
  • _territory Zielterritorium oder Table mit Territorien (Einzelnes Territorium oder Table)

Returns:

    table Liste mit Resultaten

Usage:

    local Result = API.GetEntitiesOfCategoriesInTerritories({1, 2, 3}, EntityCategories.Hero, {5, 12, 23, 24});

API.GetRelativePosition (_target, _distance, _angle, _buildingRealPos)
Errechnet eine Position relativ im angegebenen Winkel und Position zur Basisposition. Die Basis kann ein Entity oder eine Positionstabelle sein.

Alias: GetRelativePos

Parameters:

  • _target Basisposition (Skriptname, ID oder Position)
  • _distance number Entfernung
  • _angle number Winkel
  • _buildingRealPos boolean Gebäudemitte statt Gebäudeeingang

Returns:

    table Position

Usage:

    local RelativePostion = API.GetRelativePosition("pos1", 1000, 32);

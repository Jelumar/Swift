Module bundlenonplayercharacter

Mit diesem Bundle wird ein spezieller Modus für Nichtspieler-Charaktere bereitgestellt.

Die Helden eines Spielers können mit diesen NPCs sprechen. Dazu muss der Held selektiert sein. Dann kann der Spieler ihm mit einem Rechtsklick befehlen, den NPC anzusprechen. Dabei wird der Mauszeiger zu einer Hand.

Ein NPC wird durch ein leichtes Glitzern auf der Spielwelt hervorgehoben.

Anwenderfunktionen

API.NpcCompose (_Data)
Erstellt einen neuen NPC für den angegebenen Siedler.

Der NPC wird sofort aktiviert und kann angesprochen werden. Er wird durch eine glitzernde Aura hervorgehoben.

Mögliche Einstellungen für den NPC:

Eigenschaft Beschreibung
Name Stringname des Siedlers, der zum NPC werden soll.
Hero Skriptname eines Helden, mit dem der NPC sprechen will.
WrongHeroMessage Eine optionale Nachricht, wenn der falsche Held angesprochen wird.
Callback Eine Funktion, die aufgerufen wird, wenn der NPC angesprochen wird.

Alias: CreateNPC

Parameters:

  • _Data [table] Nichtspieler-Charakter

Returns:

    [table] NPC-Objekt

Usage:

    API.NpcCompose {
    Name     = "horst",
    Callback = function(_Npc, _Hero)
       -- Hier kann was passieren
    end,
}
API.NpcDispose (_Entity)
Entfernt den NPC komplett vom Entity. Das Entity bleibt dabei erhalten.

Alias: DestroyNPC

Parameters:

  • _Entity [string|number] Nichtspieler-Charakter

Usage:

    API.NpcDispose("horst")
API.NpcActivate (_Entity)
Aktiviert einen inaktiven NPC. Wenn ein NPC bereits gesprochen hat, muss er zuvor zurückgesetzt werden.

Alias: EnableNPC

Parameters:

  • _Entity [string|number] Nichtspieler-Charakter

Usage:

    API.NpcActivate("horst")
API.NpcDeactivate (_Entity)
Deaktiviert einen NPC, sodass dieser nicht angesprochen werden kann.

Alias: DisableNPC

Parameters:

  • _Entity [string|number] Nichtspieler-Charakter

Usage:

    API.NpcDeactivate("horst")
API.NpcReset (_Entity)
Setzt einen NPC zurück, sodass er nicht mehr als angesprochen gilt.

Alias: ResetNPC

Parameters:

  • _Entity [string|number] Nichtspieler-Charakter

Usage:

    API.NpcReset("horst")
API.NpcHasSpoken (_Entity)
Prüft, ob der NPC bereits angesprochen wurde. Wenn ein Ansprechpartner vorgegeben ist, muss dieser den NPC ansprechen.

Alias: TalkedToNPC

Parameters:

  • _Entity [string|number] Nichtspieler-Charakter

Returns:

    [boolean] NPC wurde angesprochen

Usage:

    API.NpcHasSpoken("horst")

Goal

Goal_NPC (_NpcName, _HeroName)
Der Held muss einen Nichtspielercharakter ansprechen.

Es wird automatisch ein NPC erzeugt und überwacht, sobald der Quest aktiviert wurde.

Parameters:

  • _NpcName [string] Skriptname des NPC
  • _HeroName [string] Skriptname des Helden (optional)

Trigger

Trigger_NPC (_NpcName, _HeroName)
Startet den Quest, sobald der NPC angesprochen wurde.

Es wird automatisch ein NPC erzeugt und überwacht, sobald der Quest erzeugt wurde.

Parameters:

  • _NpcName [string] Skriptname des NPC
  • _HeroName [string] Skriptname des Helden (optional)

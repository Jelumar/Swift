Module bundleentitycommandfunctions

Stellt Hilfsfunktionen bereit um Entities Befehle zu erteilen oder sie zu bewegen.

Das wichtigste Auf einen Blick:

  • Befehle für Entities
  • Entities bewegen
  • Entities platzieren

Anwenderfunktionen

API.CommandAttack (_Entity, _Target) Das Entity greift ein anderes Entity an, sofern möglich.
API.CommandAttackMove (_Entity, _Position) Ein Entity oder ein Battalion wird zu einer Position laufen und alle gültigen Ziele auf dem Weg angreifen.
API.CommandMove (_Entity, _Position) Bewegt das Entity zur Zielposition.
API.MoveAndLookAt (_Entity, _Position, _Distance, _moveAsEntity) Das Entity wird relativ zu einem Winkel zum Ziel bewegt und schaut es anschließend an.
API.MoveToPosition (_Entity, _Position, _Distance, _Angle, _moveAsEntity) Das Entity wird relativ zu einem Winkel zum Ziel bewegt.
API.PlaceAndLookAt (_Entity, _Position, _Distance) Das Entity wird relativ zu einem Winkel zum Zielpunkt gesetzt und schaut das Ziel an.
API.PlaceToPosition (_Entity, _Position, _Distance, _Angle) Das Entity wird relativ zu einem Winkel zum Zielpunkt gesetzt.


Anwenderfunktionen

API.CommandAttack (_Entity, _Target)
Das Entity greift ein anderes Entity an, sofern möglich.

Alias: Attack

Parameters:

  • _Entity [string|number] Angreifendes Entity
  • _Target [string|number] Angegriffenes Entity

Usage:

    API.CommandAttack("hakim", "marcus");
API.CommandAttackMove (_Entity, _Position)
Ein Entity oder ein Battalion wird zu einer Position laufen und alle gültigen Ziele auf dem Weg angreifen.

Alias: AttackMove

Parameters:

  • _Entity [string|number] Angreifendes Entity
  • _Position [string] Skriptname, EntityID oder Positionstable

Usage:

    API.CommandAttackMove("hakim", "area");
API.CommandMove (_Entity, _Position)
Bewegt das Entity zur Zielposition.

Alias: Move

Parameters:

  • _Entity [string|number] Bewegendes Entity
  • _Position [table] Positionstable

Usage:

    API.CommandMove("hakim", "pos");
API.MoveAndLookAt (_Entity, _Position, _Distance, _moveAsEntity)
Das Entity wird relativ zu einem Winkel zum Ziel bewegt und schaut es anschließend an.

Alias: MoveEx

Alias: MoveEntityFaceToFaceToAnotherOne

Parameters:

  • _Entity [string|number] Zu bewegendes Entity
  • _Position [string|number] Ziel
  • _Distance [number] Entfernung zum Ziel
  • _moveAsEntity [boolean] Blocking ignorieren

Usage:

    API.MoveAndLookAt("hakim", "saraya", 300);
API.MoveToPosition (_Entity, _Position, _Distance, _Angle, _moveAsEntity)
Das Entity wird relativ zu einem Winkel zum Ziel bewegt.

Alias: MoveEntityToPositionToAnotherOne

Parameters:

  • _Entity [string|number] Zu bewegendes Entity
  • _Position [string|number] Ziel
  • _Distance [number] Entfernung zum Ziel
  • _Angle [number] Winkel
  • _moveAsEntity [boolean] Blocking ignorieren

Usage:

    API.MoveToPosition("hakim", "saraya", 300, 0);
API.PlaceAndLookAt (_Entity, _Position, _Distance)
Das Entity wird relativ zu einem Winkel zum Zielpunkt gesetzt und schaut das Ziel an.

Alias: PlaceEntityFaceToFaceToAnotherOne

Alias: SetPositionEx

Parameters:

  • _Entity [string|number|table] Entity das bewegt wird
  • _Position [string|number|table] Position zu der bewegt wird
  • _Distance [number] Entfernung

Usage:

    API.PlaceAndLookAt("hakim", "saraya", 300);
API.PlaceToPosition (_Entity, _Position, _Distance, _Angle)
Das Entity wird relativ zu einem Winkel zum Zielpunkt gesetzt.

Alias: PlaceEntityToPositionToAnotherOne

Parameters:

  • _Entity [string|number|table] Entity das bewegt wird
  • _Position [string|number|table] Position zu der bewegt wird
  • _Distance [number] Entfernung
  • _Angle [number] Winkel

Usage:

    API.PlaceToPosition("hakim", "saraya", 300, 45);

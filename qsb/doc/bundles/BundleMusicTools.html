Bundle BundleMusicTools

Dieses Bundle bietet die Möglichkeit Musikstücke oder ganze Playlists als Stimme abzuspielen.

User Space

API.AbortMusic ()
Stopt den gerade laufenden Song und leert sowohl die Songdaten als auch die Playlist.

Alias: AbortSongOrPlaylist

API.StartMusic (_Description)
Startet ein Musikstück als Stimme.

Alias: StartMusic

Es wird nicht als Musik behandelt, sondern als Sprache! Die Lautstäkre sämtlicher Sprache wird beeinflusst, weshalb immer nur 1 Song gleichzeitig gespielt werden kann! Alle als Sprache abgespielten Sounds werden die gleiche Lautstärke haben, wie die Musik.

_Description hat folgendes Format:

  • File - Path + Dateiname
  • Volume - Lautstärke
  • length - abgespielte Länge in Sekunden (nicht zwingend |Musikstück|)
  • Fadeout - Ausblendzeit in Zehntels. (>= 5 oder 0 für sofort)
  • MuteAtmo - Hintergrundgeräusche aus
  • MuteUI - GUI-Sounds aus

Parameters:

  • _Description
API.StartMusicSimple (_File, _Volume, _Length, _FadeOut)
Vereinfachter einzeiliger Aufruf für StartSong.

Alias: StartMusicSimple

Parameters:

  • _File Pfad zur Datei
  • _Volume Lautstärke
  • _Length Abspieldower (<= Dauer Musikstück)
  • _FadeOut Ausblenden in Sekunden
API.StartPlaylist (_Playlist)
Spielt eine Playlist ab.

Alias: StartPlaylist

Eine im Skript definierte Playlist, nicht eine XML! Die Playlist kann einmal abgearbeitet oder auf Wiederholung gestellt werden. Alle Einträge haben das Format von StartSong! Zusätzlich kann der Wahrheitswert Repeat gesetzt werden, damit sich die Playlist endlos wiederholt.

Parameters:

  • _Playlist
API.StartPlaylistTitle (_Title)
Stoppt gerade gespielte Musik und startet die Playlist mit dem angegebenen Titel. Es muss eine Playlist existieren! Nachdem der Titel abgespielt ist, wird die Playlist normal weiter gespielt.

Alias: StartPlaylistTitle

Parameters:

  • _Title

Module bundleminimapmarker

Ermöglocht das Anlegen von Markierungen auf der Minimap.

Mögliche Typen von Markierungen:

  • Signal: Eine flüchtige Markierung, die nach wenigen Sekunden wieder verschwindet.
  • Marker: Eine statische Markierung, die dauerhaft verbleibt.
  • Pulse: Eine pulsierende Markierung, die dauerhaft verbleibt.

Anwenderfunktionen

API.CreateMinimapMarker (_PlayerID, _Position) Erstellt eine statische Markierung auf der Minimap.
API.CreateMinimapPulse (_PlayerID, _Position) Erstellt eine pulsierende Markierung auf der Minimap.
API.CreateMinimapSignal (_PlayerID, _Position) Erstellt eine flüchtige Markierung auf der Minimap.
API.DestroyMinimapSignal (_ID) Zerstört eine Markierung auf der Minimap.

Module bundleconstructioncontrol

Mit diesem Bundle kann der Bau von Gebäudetypen oder Gebäudekategorien unterbunden werden.

Verbote können für bestimmte Bereiche (kreisförmige Gebiete um ein Zentrum) oder ganze Territorien vereinbart werden.

Das wichtigste Auf einen Blick:

  • Den Bau von Gebäuden an bestimmten Orten steuern.
    Bau verbieten, Bau erlauben

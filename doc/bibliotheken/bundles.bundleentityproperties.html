Module bundles.bundleentityproperties

In diesem Bundle wird eine Klasse bereitgestellt, die alle wichtigen Abfragen an ein Entity in sich vereint.

Mit diesem Properties Wrapper kannst Du bequem die Eigenschaften von Entities abfragen und ändern.

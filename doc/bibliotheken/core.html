Anwenderfunktionen

API.ActivateIO (_ScriptName, _State)
Aktiviert ein interaktives Objekt, sodass es benutzt werden kann.

Der State bestimmt, ob es immer aktiviert werden kann, oder ob der Spieler einen Helden benutzen muss. Wird der Parameter weggelassen, muss immer ein Held das Objekt aktivieren.

Alias: InteractiveObjectActivate

Parameters:

  • _ScriptName [string] Skriptname des IO
  • _State [number] Aktivierungszustand

Usage:

    API.ActivateIO("Haus1", 0)
API.ActivateIO("Hut1")
API.AddHotKey (_Key, _Description)
Fügt eine Beschreibung zu einem selbst gewählten Hotkey hinzu.

Ist der Hotkey bereits vorhanden, wird -1 zurückgegeben.

Parameters:

  • _Key [string] Tastenkombination
  • _Description [string] Beschreibung des Hotkey

Returns:

    [number] Index oder Fehlercode

API.Install ()
Initialisiert alle verfügbaren Bundles und führt ihre Install-Methode aus.

Bundles werden immer getrennt im globalen und im lokalen Skript gestartet. Diese Funktion muss zwingend im globalen und lokalen Skript ausgeführt werden, bevor die QSB verwendet werden kann.

API.InstanceTable (_Source, _Dest)
Kopiert eine komplette Table und gibt die Kopie zurück. Tables können nicht durch Zuweisungen kopiert werden. Verwende diese Funktion. Wenn ein Ziel angegeben wird, ist die zurückgegebene Table eine Vereinigung der 2 angegebenen Tables. Die Funktion arbeitet rekursiv.

Alias: CopyTableRecursive

Parameters:

  • _Source [table] Quelltabelle
  • _Dest [table] (optional) Zieltabelle

Returns:

    [table] Kopie der Tabelle

Usage:

    Table = {1, 2, 3, {a = true}}
Copy = API.InstanceTable(Table)

API.RestartQuest (_QuestName, _Quiet)
Startet den Quest neu.

Der Quest muss beendet sein um ihn wieder neu zu starten. Wird ein Quest neu gestartet, müssen auch alle Trigger wieder neu ausgelöst werden, außer der Quest wird manuell getriggert.

Alle Änderungen an Standardbehavior müssen hier berücksichtigt werden. Wird ein Standardbehavior in einem Bundle verändern, muss auch diese Funktion angepasst oder überschrieben werden.

Alias: RestartQuestByName

Parameters:

  • _QuestName [string] Name des Quest
  • _Quiet [boolean] Keine Meldung anzeigen
API.SendCart (_position, _player, _good, _amount, _cartOverlay, _ignoreReservation)
Sendet einen Handelskarren zu dem Spieler. Startet der Karren von einem Gebäude, wird immer die Position des Eingangs genommen.

Alias: SendCart

Parameters:

  • _position [string|number] Position
  • _player [number] Zielspieler
  • _good [number] Warentyp
  • _amount [number] Warenmenge
  • _cartOverlay [number] (optional) Overlay für Goldkarren
  • _ignoreReservation [boolean] (optional) Marktplatzreservation ignorieren

Returns:

    number: Entity-ID des erzeugten Wagens

Usage:

    -- API-Call
API.SendCart(Logic.GetStoreHouse(1), 2, Goods.G_Grain, 45)
-- Legacy-Call mit ID-Speicherung
local ID = SendCart("Position_1", 5, Goods.G_Wool, 5)

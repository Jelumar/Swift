Module addons.addoncastlestore

Dieses Bundle stellt ein Burglager zur Verfügung, das sich ähnlich wie das normale Lager verhält.

Das Burglager ist von der Ausbaustufe der Burg abhängig. Je weiter die Burg ausgebaut wird, desto höher ist das Limit. Eine Ware wird dann im Burglager eingelagert, wenn das eingestellte Limit der Ware im Lagerhaus erreicht wird.

Der Spieler kann das allgemeine Verhalten des Lagers für alle Waren wählen und zusätzlich für einzelne Waren andere Verhalten bestimmen. Waren können eingelagert und ausgelagert werden. Eingelagerte Waren können zusätzlich gesperrt werden. Eine gesperrte Ware wird nicht wieder ausgelagert, auch wenn Platz im Lager frei wird.

Muss ein Spieler einen Tribut aus dem lagerhaus begleichen, eine bestimmte Menge an Waren erreichen oder die Kosten Zur aktivierung eines interaktien Objektes bezahlen, werden die Güter im Burglager automatisch mit einbezogen, wenn sie nicht gesperrt wurden.

Das wichtigste Auf einen Blick:

  • Burglager in der Burg anlegen
  • Warenmenge in der Burg abfragen
  • Waren dem Burglager hinzufügen
  • Waren aus der Burg entfernen

Anwenderfunktionen

API.CastleStoreAddGood (_PlayerID, _Good, _Amount) Fügt dem Burglager des Spielers eine Menga an Waren hinzu.
API.CastleStoreCountGood (_PlayerID, _Good) Gibt die Menge an Waren des Typs im Burglager des Spielers zurück.
API.CastleStoreCreate (_PlayerID) Erstellt ein Burglager für den angegebenen Spieler.
API.CastleStoreDestroy (_PlayerID) Zerstört das Burglager des angegebenen Spielers.
API.CastleStoreRemoveGood (_PlayerID, _Good, _Amount) Entfernt eine Menge von Waren aus dem Burglager des Spielers.

API.CastleStoreAddGood (_PlayerID, _Good, _Amount)
Fügt dem Burglager des Spielers eine Menga an Waren hinzu.

Parameters:

  • _PlayerID [number] ID des Spielers
  • _Good [number] Typ der Ware
  • _Amount [number] Menge der Ware

Usage:

    API.CastleStoreAddGood(1, Goods.G_Wood, 50);

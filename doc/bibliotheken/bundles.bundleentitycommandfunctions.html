Module bundles.bundleentitycommandfunctions

Stellt Hilfsfunktionen bereit um Entities Befehle zu erteilen oder sie zu bewegen.

Das wichtigste Auf einen Blick:

  • Befehle für Entities
  • Entities bewegen
  • Entities platzieren

Anwenderfunktionen

API.CommandAttack (_Entity, _Target) Das Entity greift ein anderes Entity an, sofern möglich.
API.CommandAttackMove (_Entity, _Position) Ein Entity oder ein Battalion wird zu einer Position laufen und alle gültigen Ziele auf dem Weg angreifen.
API.CommandMove (_Entity, _Position) Bewegt das Entity zur Zielposition.
API.MoveAndLookAt (_Entity, _Position, _Distance, _moveAsEntity) Das Entity wird relativ zu einem Winkel zum Ziel bewegt und schaut es anschließend an.
API.MoveToPosition (_Entity, _Position, _Distance, _Angle, _moveAsEntity) Das Entity wird relativ zu einem Winkel zum Ziel bewegt.
API.PlaceAndLookAt (_Entity, _Position, _Distance) Das Entity wird relativ zu einem Winkel zum Zielpunkt gesetzt und schaut das Ziel an.
API.PlaceToPosition (_Entity, _Position, _Distance, _Angle) Das Entity wird relativ zu einem Winkel zum Zielpunkt gesetzt.

Module bundles.bundleentityselection

Mit diesem Bundle kann die Selektion von Entities gesteuert werden.

Das wichtigste auf einen Blick:

  • Einheiten entlassen
  • Selektierte Entities ermitteln

Anwenderfunktionen

API.DisableRefillTrebuchet (_Flag) Deaktiviert oder aktiviert das Nachfüllen von Trebuchets.
API.DisableReleaseSiegeEngines (_Flag) Deaktiviert oder aktiviert das Entlassen von Kriegsmaschinen.
API.DisableReleaseSoldiers (_Flag) Deaktiviert oder aktiviert das Entlassen von Soldaten.
API.DisableReleaseThieves (_Flag) Deaktiviert oder aktiviert das Entlassen von Dieben.

API.GetSelectedEntity () Gibt die ID des selektierten Entity zurück.

Gibt alle selektierten Entities zurück.

Returns:

Usage:

API.GetSelectedEntity ()
Gibt die ID des selektierten Entity zurück.

Wenn mehr als ein Entity selektiert sind, wird das erste Entity zurückgegeben. Sind keine Entities selektiert, wird 0 zurückgegeben.

Returns:

    [number] ID des selektierten Entities

Usage:

    local SelectedEntity = API.GetSelectedEntity();

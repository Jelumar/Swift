Module bundles.bundleentityscriptingvalues

Mit diesem Bundle können Eigenschaften von Entities abgefragt oder verändert werden, die sonst unzugänglich wären.

Dazu zählen beispielsweise die Größe des Entity und das Bewegungsziel.

API.GetEntityScale (_Entity)
Gibt den Größenfaktor des Entity zurück.

Der Faktor gibt die relative Größe des Entity zu seiner normalen Größe an.

Alias: GetScale

Parameters:

  • _Entity [string|number] Entity

Returns:

    [number] Größenfaktor

Usage:

    local Scale = API.GetEntityScale("alandra")
API.GetMovementTarget (_Entity)
Gibt die Position zurück, zu der sich das Entity bewegt.

Über diese Koordinaten könnte man prüfen, ob ein Entity sich in einen Bereich bewegt, in dem es nichts zu suchen hat.

Alias: GetMovingTarget

Parameters:

  • _Entity [string|number] Entity

Returns:

    [table] Positionstabelle

Usage:

    -- Hakim bleibt stehen, wenn er in ein Sperrgebiet bewegt wird.
local Destination = API.GetMovementTarget("hakim");
if GetDistance(Destination, "LockedArea") < 2000 then
    local x,y,z = Logic.EntityGetPos(GetID("hakim"));
    Logic.DEBUG_SetSettlerPosition(GetID("hakim"), x, y):
end
API.IsActiveNpc (_Entity)
Gibt zurück, ob das NPC-Flag bei dem Siedler gesetzt ist.

Auf diese Weise kann geprüft werden, ob ein NPC auf dem Entity aktiv ist.

Alias: IsNpc

Parameters:

  • _Entity [string|number] Entity

Returns:

    [boolean] Ist NPC

Usage:

    local Active = API.IsActiveNpc("alandra");
if Active then
    API.Note("NPC is active");
end

API.SetEntityPlayer (_Entity, _PlayerID)
Erzwingt einen neuen Besitzer für das Entity.

Mit dieser Funktion werden die Sicherungen des Spiels umgangen! Es ist möglich ein Raubtier einem richtigen Spieler zuzuweisen.

Alias: ChangePlayer

Parameters:

  • _Entity [string|number] Entity
  • _PlayerID [number] Besitzer
API.SetEntityScale (_Entity, _Scale)
Setzt den Größenfaktor des Entity.

Bei einem Siedler wird ebenfalls versucht die Bewegungsgeschwindigkeit an die Größe anzupassen, was aber nicht bei allen Siedlern möglich ist.

Alias: SetScale

Parameters:

  • _Entity [string|number] Entity
  • _Scale [number] Größenfaktor

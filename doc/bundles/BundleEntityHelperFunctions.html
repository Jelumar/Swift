API.MoveToPosition (_Entity, _Position, _Distance, _Angle, _moveAsEntity)
Das Entity wird zum ziel bewegt und kann relativ um das Ziel in einem Winkel bewegt werden. Das Entity wird das Ziel anschießend anschauen. Die Funktion kann auch Schiffe bewegen, indem der letzte Parameter true gesetzt wird.

Alias: MoveEx

Parameters:

  • _Entity Zu bewegendes Entity
  • _Position Ziel
  • _Distance Entfernung zum Ziel
  • _Angle Winkel
  • _moveAsEntity Blocking ignorieren

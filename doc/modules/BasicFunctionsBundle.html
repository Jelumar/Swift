Module BasicFunctionsBundle

Enthält grundlegende Hilfsfunktionen, auf die alle anderen Bundles immer zugreifen können.

Functions

GetQuestID (_Questname) Gibt die ID des Quests mit dem angegebenen Namen zurück.
IsValidQuestID (_QuestID) Prüft, ob die ID zu einem Quest gehört bzw.
AddQuestBehavior (_Behavior) Fügt ein Behavior der Liste im Assistenten hinzu und erzeugt einen Konstruktor, der die Behavior-Table zurückgibt.
RemoveQuestBehavior (_Name) Macht ein Behavior unzugänglich.
IsValidQuestName (_Name) Prüft, ob der Questname verbotene Zeichen enthält.

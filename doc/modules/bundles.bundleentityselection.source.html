Module bundles.bundleentityselection.source

Mit diesem Bundle kann die Selektion von Entities gesteuert werden.

Anwenderfunktionen

API.DisableMilitaryRelease (_Flag) Deaktiviert oder aktiviert das Entlassen von Soldaten.
API.DisableRefillTrebuchet (_Flag) Deaktiviert oder aktiviert das Nachfüllen von Trebuchets.
API.DisableSiegeEngineRelease (_Flag) Deaktiviert oder aktiviert das Entlassen von Kriegsmaschinen.
API.DisableThiefRelease (_Flag) Deaktiviert oder aktiviert das Entlassen von Dieben.
API.GetSelectedEntities () Gibt alle selektierten Entities zurück.
API.GetSelectedEntity () Gibt die ID des selektierten Entity zurück.
API.IsEntityInSelection (_Entity) Prüpft ob das Entity selektiert ist.

Internal

BundleEntitySelection.Global:DeactivateRefillTrebuchet (_Boolean) Deaktiviert oder aktiviert das Nachfüllen von Trebuchets.
BundleEntitySelection.Global:Install () Initialisiert das Bundle im globalen Skript.
BundleEntitySelection.Global:MilitaryCallForRefiller (_EntityID) Erzeugt einen Wagen, der zu dem Trebuchet fährt und es auffüll.
BundleEntitySelection.Global:MilitaryDisambleTrebuchet (_EntityID) Baut ein Trebuchet zu einem Trebuchet-Wagen ab.
BundleEntitySelection.Global:MilitaryErectTrebuchet (_EntityID) Baut einen Trebuchet-Wagen zu einem Trebuchet aus.
BundleEntitySelection.Local.OnSelectionCanged (_Source) Callback-Funktion, die aufgerufen wird, wenn sich die Selektion ändert.
BundleEntitySelection.Local:DeactivateRefillTrebuchet (_Boolean) Deaktiviert oder aktiviert das Nachfüllen von Trebuchets.
BundleEntitySelection.Local:ExtendedLeaderSortOrder () Erzeugt die erweiterte Selektion mit Munitionswagen und Trebuchets.
BundleEntitySelection.Local:Install () Initialisiert das Bundle im lokalen Skript.
BundleEntitySelection.Local:NormalLeaderSortOrder () Erzeugt die normale Sortierung ohne Munitionswagen und Trebuchets.
BundleEntitySelection.Local:OverwriteGetStringTableText () Überscheibt die Funktion, die die Ingame-Texte aus den Quellen ausließt, sodass eigene Texte für Keys angezeigt werden.
BundleEntitySelection.Local:OverwriteMilitaryCommands () Überschreibt die Millitärkommandos "Stop" und "Angreifen".
BundleEntitySelection.Local:OverwriteMilitaryDisamble () Überschreibt das Abbauen von Kriegsmaschinen, sodass auch Trebuchets abgebaut werden können.
BundleEntitySelection.Local:OverwriteMilitaryDismount () Überschreibt die Funktion zur Beendigung der Eskorte, damit Einheiten auch entlassen werden können.
BundleEntitySelection.Local:OverwriteMilitaryErect () Überschreibt das Aufbauen von Kriegsmaschinen, sodass auch Trebuchets auf- und abgebaut werden können.
BundleEntitySelection.Local:OverwriteMultiselectIcon () Überschreibt die Multiselektion, damit Trebuchets ein Icon bekommen.
BundleEntitySelection.Local:OverwriteNamesAndDescription () Hängt eine Funktion an die GUI_Tooltip.SetNameAndDescription an, sodass Tooltips überschrieben werden können.
BundleEntitySelection.Local:OverwriteSelectAllUnits () Überschreibt die Militärselektion, sodass der Spieler mit SHIFT zusätzlich die Munitionswagen und Trebuchets selektieren kann.
BundleEntitySelection.Local:OverwriteSelectKnight () Überschreibt den SelectKnight-Button.
BundleEntitySelection.Local:OverwriteThiefDeliver () Überschreibt "Beute abließern", sodass Diebe entlassen werden können.
BundleEntitySelection.Local:SetTooltip (_TitleText, _DescText) Schreibt einen anderen Text in einen normalen Tooltip.

API.GetSelectedEntity ()
Gibt die ID des selektierten Entity zurück.

Wenn mehr als ein Entity selektiert sind, wird das erste Entity zurückgegeben. Sind keine Entities selektiert, wird 0 zurückgegeben.

Returns:

    [number] ID des selektierten Entities

Usage:

    local SelectedEntity = API.GetSelectedEntity();

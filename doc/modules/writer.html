Module writer

Läd die Quelldateien und fügrt sie zur QSB zusammen.

Dieses Skript kann nicht aus dem Spiel ausgeführt werden! Es wird eine Installation von Lua 5.1 oder höher auf dem PC benötigt. Lua kann auf Sorceforge herruntergeladen werden.
Lua-Release

Um Lua zu installieren Lege die Quellen von Lua unter Windows nach C:/Lua und passe die Systemvariable PATH an. Anschließend benötigst du eine Konsole mit Linux-Befehlen. Dazu kannst du dir z.B. Git-Bash installieren. Alle Skripte sollten nun problemlos funktionieren.

Unter Linux und Mac müssen einige Abhängigkeiten von LDoc nachinstalliert werden um die Dokumentation generieren zu können. Die Installationen müssen mit Root-Rechten durchgeführt werden!

  • apt install luarocks
  • luarocks install luafilesystem
  • luarocks install penlight

Um die QSB zusammenfügen zu lassen, nutze die make im Hauptverzeichnis des Projektes oder rufe dieses Skript in der Shell auf.

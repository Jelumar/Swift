API.ChangeTerrainHeightInSquare (_Center, _Offset, _Hight, _Relative)
Ändert die Landhöhe in einem Quadrat. Offset bestimmt die Abstände der Eckpunkte zum Zentralpunkt.

Wird die relative Höhe verwendet, wird die Landhöhe nicht absolut gesetzt sondern von der aktuellen Landhöhe ausgegangen. Das Land muss nicht eben sein. Auf diese Weise können Strukturen unverändert angehoben werden.

Alias: TerrainHeight

Parameters:

  • _Center [string|number] Zentralpunkt
  • _Offset [number] Entfernung der Ecken zum Zentrum
  • _Hight [number] Neue Höhe
  • _Relative [boolean] Relative Höhe benutzen

Usage:

    API.ChangeTerrainHeightInSquare("area", 500, 5555, true);
API.ChangeTerrainTypeInSquare (_Center, _Offset, _TerrainType)
Ändert die Bodentextur innerhalb des Quadrates. Offset bestimmt die Abstände der Eckpunkte zum Zentralpunkt.

Hinweis: Für weitere Informationen zu Terraintexturen siehe https://siedelwood-neu.de/23879-2/

Alias: TerrainType

Parameters:

  • _Center [string|number] Zentralpunkt
  • _Offset [number] Entfernung der Ecken zum Zentrum
  • _TerrainType [number] Textur ID

Usage:

    API.ChangeTerrainTypeInSquare("area", 500, 48)
API.ChangeWaterHeightInSquare (_Center, _Offset, _Hight, _Relative)

Alias: WaterHeight

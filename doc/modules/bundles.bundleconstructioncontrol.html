Module bundles.bundleconstructioncontrol

Ermöglicht es Gebiete oder Territorien auf der Map zu definieren, auf der ein Gebäude nicht gebaut oder nicht abgerissen werden darf.

Das wichtigste Auf einen Blick:

  • Den Abriss für bestimmte Entities steuern.
    Entities beschützen, Schutz aufheben
  • Den Bau von Gebäuden an bestimmten Orten steuern.
    Bau verbieten, Bau erlauben

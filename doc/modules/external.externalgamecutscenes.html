Module external.externalgamecutscenes

DEPRECATED

Dieses Bundle verwaltet den Aufruf der mit dem SCA-Tool erstellten Cutscenes.

Ausserdem ermöglicht es das direkte Erfassen der Kamerapositionen aus der Map heraus.

Anwenderfunktionen

CS.CreateCutsceneRealtime_DEV_ONLY () Erstellen einer Echtzeit Cutscene, um diese mit dem SCA Tool in eine cs Datei umzuwandeln
CS.CreateCutscene_DEV_ONLY () Erstellen einer Cutscene, um diese mit dem SCA Tool in eine cs Datei umzuwandeln
CS.IsCutsceneActive () Prüft, ob eine Cutscene aktiv ist.
CS.StartCutscene (_cutscene) Startet eine Cutscene.

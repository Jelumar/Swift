Module bundles.bundlefollowknight

Dieses Bundle bietet Funktionalität um Entities dauerhaft einem Helden (bzw Ritter) folgen zu lassen.

Anwenderfunktionen

API.FollowKnightSaveStart (_Entity, _Knight, _Distance, _Angle) Lässt einen Siedler einem Helden folgen.
API.FollowKnightSaveStop (_JobID) Beendet einen Verfolgungsjob.

Anwenderfunktionen

API.FollowKnightSaveStart (_Entity, _Knight, _Distance, _Angle)
Lässt einen Siedler einem Helden folgen. Gibt die ID des Jobs zurück, der die Verfolgung steuert.

Hinweis: Wenn eines der Entities zerstört wird, oder ins Koma fällt, wird der Job beendet!

Alias: AddFollowKnightSave

Parameters:

  • _Entity Entity das folgt (skriptname oder ID)
  • _Knight Held (Skriptname oder ID)
  • _Distance number Entfernung, die uberschritten sein muss
  • _Angle number Ausrichtung

Returns:

    number Job-ID

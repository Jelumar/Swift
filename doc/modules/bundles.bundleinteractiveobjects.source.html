Module bundles.bundleinteractiveobjects.source

Interaktive Objekte sind Gegenstände auf der Karte, mit denen interagiert werden kann.

Diese Interaktion geschieht über einen Button. Ziel dieses Bundels ist es, die funktionalität von interaktiven Objekten zu erweitern. Es ist möglich, beliebige Objekte zu interaktiven Objekten zu machen.

Die Einsatzmöglichkeiten sind vielfältig. Wenn ein Gegenstand oder ein Objekt mit einer Funktion versehen ist, kann dies in verschiedenem Kontext an die Geschichte angepasst werden: z.B. Helbel öffnen eine Geheimtür, ein Gegenstand wird vom Helden aufgehoben, ein Marktstand, der etwas verkauft, ....

Das wichtigste Auf einen Blick:

  • Objekt erzeugen
  • Ein- und ausschalten von interaktiven Objekten
  • Anzeigenamen im Questfenster definieren

API.AddCustomIOName (_Key, _Text)
Erzeugt eine Beschriftung für Custom Objects.

Im Questfenster werden die Namen von Cusrom Objects als ungesetzt angezeigt. Mit dieser Funktion kann ein Name angelegt werden.

Alias: AddCustomIOName

Parameters:

  • _Key [string] Typname des Entity
  • _Text [string] Text der Beschriftung

Usage:

    API.AddCustomIOName("D_X_ChestClosed", "Schatztruhe");
API.AddCustomIOName("D_X_ChestOpenEmpty", "Leere Schatztruhe");
API.CreateObject (_Description)
Erzeugt ein interaktives Objekt.

Die Parameter des interaktiven Objektes werden durch seine Beschreibung festgelegt. Die Beschreibung ist eine Table, die bestimmte Werte für das Objekt beinhaltet. Dabei müssen nicht immer alle Werte angegeben werden.

Mögliche Angaben:

Feldname Beschreibung Optional
Name Der Skriptname des Entity, das zum interaktiven Objekt wird. nein
Title Der angezeigter Name im Beschreibungsfeld. ja
Text Der Beschreibungstext, der im Tooltip angezeigt wird. ja
Texture Bestimmt die Icongrafik, die angezeigt wird. Dabei kann es sich um eine Ingame-Grafik oder eine eigene Grafik halten. ja
Distance Die minimale Entfernung zum Objekt, die ein Held benötigt um das objekt zu aktivieren. ja
Waittime Die Zeit, die ein Held benötigt, um das Objekt zu aktivieren. Die Wartezeit ist nur für I_X_ Entities verfügbar. ja
Costs Eine Table mit dem Typ und der Menge der Kosten. ja
Reward Der Warentyp und die Menge der gefundenen Waren im Objekt. ja
Callback Eine Funktion, die ausgeführt wird, sobald das Objekt aktiviert wird. ja
Condition Eine Funktion, die vor der Aktivierung eine Beringung prüft. ja
ConditionUnfulfilled Eine Nachricht, die angezeigt wird, falls die Bedingung nicht erfüllt ist. ja
Opener Ein spezieller Held, der als einziger das Objekt aktivieren kann. ja
WrongKnight Nachricht, die angezeigt wird, wenn der falsche Held das Objekt aktivieren will. ja
State Bestimmt, wie sich der Button des interaktiven Objektes verhält. ja

Zusätzlich können beliebige weitere Felder an das Objekt angehangen werden. Sie sind ausnahmslos im Callback und in der Condition des Objektes abrufbar.

Alias: CreateObject

Parameters:

  • _Description [table] Beschreibung

Usage:

    -- Ein einfaches Objekt erstellen:
CreateObject {
    Name     = "hut",
    Distance = 1500,
    Callback = function(_Data)
        API.Note("Do something...");
    end,
}
API.InteractiveObjectActivate (_EntityName, _State)
Aktiviert ein Interaktives Objekt, sodass es vom Spieler aktiviert werden kann.

Der State bestimmt, ob es immer aktiviert werden kann, oder ob der Spieler einen Helden benutzen muss. Wird der Parameter weggelassen, muss immer ein Held das Objekt aktivieren.

Alias: InteractiveObjectActivate

Parameters:

  • _EntityName [string] Skriptname des Objektes
  • _State [number] State des Objektes
API.InteractiveObjectDeactivate (_EntityName)
Deaktiviert ein interaktives Objekt, sodass es nicht mehr vom Spieler benutzt werden kann.

Alias: InteractiveObjectDeactivate

Parameters:

  • _EntityName [string] Scriptname des Objektes
API.RemoveInteractiveObject (_EntityName)
Löscht ein interaktives Objekt.

Das Entity wird dabei nicht gelöscht. Es wird ausschließlich die Konfiguration des Objektes entfernt.

Alias: RemoveInteractiveObject

Parameters:

  • _EntityName [string] Skriptname des IO

  • _Description [table] Beschreibung

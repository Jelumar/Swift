Module bundles.bundlestockbreeding

Ermöglicht die Aufzucht von Nutzrieren wie Schafe und Kühe durch den Spieler.

Kosten für die Aufzucht oder die benötigte Menge an Tieren um mit der Zucht zu beginnen, sind frei konfigurierbar.

Anwenderfunktionen

API.SetCattleGrainCost (_Amount) Mit dieser Funktion werden die Getreidekosten für die Aufzucht von Kühen festgelegt.
API.SetCattleNeeded (_Amount) Legt die Anzahl an Kühen fest, die zur Zucht eines neuen Trieres mindestens in einem Gatter vorhanden sein müssen.
API.SetSheepGrainCost (_Amount) Mit dieser Funktion werden die Getreidekosten für die Aufzucht von Schafen festgelegt.
API.SetSheepNeeded (_Amount) Legt die Anzahlö an Schafen fest, die zur Zucht eines neuen Trieres mindestens in einem Gatter vorhanden sein müssen.
API.UseBreedCattle (_flag) Erlaube oder verbiete dem Spieler Kühe zu züchten.
API.UseBreedSheeps (_flag) Erlaube oder verbiete dem Spieler Schafe zu züchten.

Internal

BundleStockbreeding.Local:BuyAnimal (_eID) Diese Funktion erzeugt ein Nutztier und entfernt das Getreide vom Spieler.
BundleStockbreeding.Local:Install () Initalisiert das Bundle im lokalen Skript.

BundleStockbreeding.Local:TextCosts (_Title, _Text, _DisabledText, _Costs, _InSettlement) Setzt den Kostentooltip des aktuellen Widgets.


Anwenderfunktionen

API.SetCattleGrainCost (_Amount)
Mit dieser Funktion werden die Getreidekosten für die Aufzucht von Kühen festgelegt.

Will der Spieler nun eine Kuh züchten, muss er mindestens die angegebene Menge an Getreide besitzen. Das Getreide wird dann aus dem Lager entfernt.

Alias: SetCattleGrainCost

Parameters:

  • _Amount number Getreidekosten

Usage:

    -- Wucherpreise zum Züchten!
API.SetCattleGrainCost(50);
API.SetCattleNeeded (_Amount)
Legt die Anzahl an Kühen fest, die zur Zucht eines neuen Trieres mindestens in einem Gatter vorhanden sein müssen.

Alias: SetCattleNeeded

Parameters:

  • _Amount number Benötigte Menge

Usage:

    -- Es werden keine Kühe zur Zucht benötigt:
API.SetCattleNeeded(0);
API.SetSheepGrainCost (_Amount)
Mit dieser Funktion werden die Getreidekosten für die Aufzucht von Schafen festgelegt.

Will der Spieler nun ein Schaf züchten, muss er mindestens die angegebene Menge an Getreide besitzen. Das Getreide wird dann aus dem Lager entfernt.

Alias: SetSheepGrainCost

Parameters:

  • _Amount number Getreidekosten

Usage:

    -- Wucherpreise zum Züchten!
API.SetSheepGrainCost(50);
API.SetSheepNeeded (_Amount)
Legt die Anzahlö an Schafen fest, die zur Zucht eines neuen Trieres mindestens in einem Gatter vorhanden sein müssen.

Alias: SetSheepNeeded

Parameters:

  • _Amount number Benötigte Menge

Usage:

    -- Es wird ein volles Gatter zur Zucht benötigt:
API.SetSheepNeeded(5);
API.UseBreedCattle (_flag)
Erlaube oder verbiete dem Spieler Kühe zu züchten.

Wenn der Spieler keine Kühe züchten soll, kann ihm dieses Recht durch diese Funktion genommen werden. Natürlich kann das Recht auf diesem Weg auch wieder zurückgegeben werden.

Alias: UseBreedCattle

Parameters:

  • _flag boolean Kuhzucht aktiv/inaktiv

Usage:

    -- Es können keine Kühe gezüchtet werden
API.UseBreedCattle(false);
API.UseBreedSheeps (_flag)
Erlaube oder verbiete dem Spieler Schafe zu züchten.

Wenn der Spieler keine Schafe züchten soll, kann ihm dieses Recht durch diese Funktion genommen werden. Natürlich kann das Recht auf diesem Weg auch wieder zurückgegeben werden.

Alias: UseBreedSheeps

Parameters:

  • _flag boolean Schafzucht aktiv/inaktiv

Usage:

    -- Schafsaufzucht ist erlaubt
API.UseBreedSheeps(true);

Module QsbFramework

Das Framework läd die einzelnen Bundles und beinhaltet alle geteilte Funktionen.

Interne Spielfunktionen, die überschrieben werden, müssen ebenfalls hier definiert werden.

API.ValidQuestName (_Name)
Prüft, ob der Questname formal korrekt ist. Questnamen dürfen i.d.R. nur die Zeichen A-Z, a-7, 0-9, - und _ enthalten.

Alias: ValidQuestName

Parameters:

  • _Name Quest

Returns:

    boolean
API.InstanceTable (_Source, _Dest)
Kopiert eine komplette Table und gibt die Kopie zurück. Wenn ein Ziel angegeben wird, ist die zurückgegebene Table eine vereinigung der 2 angegebenen Tables. Die Funktion arbeitet rekursiv und ist für beide Arten von Index. Die Funktion kann benutzt werden, um Klassen zu instanzieren.

Alias: CopyTableRecursive

Parameters:

  • _Source Quelltabelle
  • _Dest Zieltabelle

Returns:

    table
API.TraverseTable (_Data, _Table)
Sucht in einer Table nach einem Wert. Das erste Aufkommen des Suchwerts wird als Erfolg gewertet.

Alias: Inside

Parameters:

  • _Data Datum, das gesucht wird
  • _Table Tabelle, die durchquert wird

Returns:

    boolean
